1. What is HTML? 🌐

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses tags to structure content and tell browsers how to display information.

Key Points:

  • HTML is not a programming language - it's a markup language
  • It uses tags (like <h1>, <p>) to structure content
  • Every HTML document needs a proper structure
  • It works with CSS (styling) and JavaScript (functionality)

2. Basic HTML Structure 🏗️

Every HTML document follows this basic structure:

Essential HTML Elements:

  • <!DOCTYPE html> - Tells the browser this is HTML5
  • <html> - Root element of the page
  • <head> - Contains metadata (title, styles, scripts)
  • <body> - Contains the visible content

4. Lists 📋

HTML supports two main types of lists:

Unordered List (Bullet Points):

  • First item
  • Second item
  • Third item
    • Nested item 1
    • Nested item 2

Ordered List (Numbered):

  1. First step
  2. Second step
  3. Third step

List Tags:

  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item (used in both types)

5. Links 🔗

Links connect different pages and resources:

Different Types of Links:

  • External link (opens in new tab)
  • Internal link (jumps to section on same page)
  • Email link
  • Phone link

Link Attributes:

  • href - The destination URL
  • target="_blank" - Opens in new tab
  • title - Tooltip text on hover

7. Tables 📊

Tables organize data in rows and columns:

HTML Tag Purpose Example
<table> Creates a table Container for all table elements
<tr> Table row Each row of data
<th> Table header Column titles (bold by default)
<td> Table data Individual cells

Table Structure:

Tables use a hierarchical structure: table → thead/tbody → tr → th/td

9. Semantic HTML 🏷️

Semantic HTML uses meaningful tags that describe their content:

Semantic Elements:

  • <header> - Page or section header
  • <nav> - Navigation links
  • <main> - Main content area
  • <section> - Thematic grouping of content
  • <article> - Self-contained content
  • <aside> - Sidebar content
  • <footer> - Page or section footer

Benefits of Semantic HTML:

  • Better accessibility for screen readers
  • Improved SEO (Search Engine Optimization)
  • Clearer code structure
  • Easier maintenance

10. Basic Styling 🎨

While HTML structures content, CSS styles it. This page uses internal CSS in the <style> tag.

CSS Basics:

  • Selectors: Target HTML elements (h1, .class, #id)
  • Properties: What to style (color, font-size, margin)
  • Values: How to style it (red, 16px, 20px)

Three Ways to Add CSS:

  1. Inline: <h1 style="color: red;">
  2. Internal: <style> tag in <head>
  3. External: Separate .css file linked with <link>
